Logical Connectives

Term Description
Statements/Propositions Sentences that are true or false but not both.
Conjunction (AND) Both statements must be true for the result to be true.
Disjunction (OR) At least one of the statements must be true for the result to be true.
Implication (IF-THEN) If the first statement is true, then the second one must be true.
Biconditional (IF AND ONLY IF) Both statements are true or both are false.
Tautology A statement that is always true, no matter what.
Contradiction A statement that is always false, no matter what.
