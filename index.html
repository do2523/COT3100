Logical Connectives

Term Description
Statements/Propositions Sentences that are true or false but not both.
Conjunction (AND) Both statements must be true for the result to be true.
Disjunction (OR) At least one of the statements must be true for the result to be true.
Implication (IF-THEN) If the first statement is true, then the second one must be true.
Biconditional (IF AND ONLY IF) Both statements are true or both are false.
Tautology A statement that is always true, no matter what.
Contradiction A statement that is always false, no matter what.

    Rules of Inference

    Rule Name Symbolic Form Explanation
    Modus Ponens p → q, p ⊢ q If p is true and p → q is true, then q must be true.
    Modus Tollens p → q, ¬q ⊢ ¬p If p → q is true and q is false, then p must be false.
    Hypothetical Syllogism p → q, q → r ⊢ p → r If p → q and q → r, then p → r.
    Disjunctive Syllogism p ∨ q, ¬p ⊢ q If p ∨ q is true and p is false, then q must be true.
    Addition p ⊢ p ∨ q If p is true, then p ∨ q is true for any q.
    Simplification p ∧ q ⊢ p If p ∧ q is true, then p is true.
    Conjunction p, q ⊢ p ∧ q If p is true and q is true, then p ∧ q is true.
    Resolution p ∨ q, ¬p ∨ r ⊢ q ∨ r If p ∨ q and ¬p ∨ r are true, then q ∨ r is true.
